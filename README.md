# Inteligencia-Artificial
Este repositorio contiene la colección de prácticas y ejercicios desarrollados en Google Colab para la asignatura de *Inteligencia Artificial* de la *Universidad Nacional Rosario Castellanos*. El objetivo es documentar la implementación y el entendimiento de los principales modelos de redes neuronales, desde los perceptrones básicos hasta arquitecturas modernas.

El contenido sigue la estructura del curso cubriendo: *Fundamentos de IA, Arquitecturas Clásicas y Arquitecturas Modernas de Redes Neuronales.*

## Contenido por Sección

### **Convolucionales/**
Contiene notebooks y código relacionados con redes convolucionales (CNN). Útil para tareas como clasificación de imágenes, reconocimiento de patrones espaciales, etc.

### **Multicapa/**
Se incluyen modelos basados en perceptrones multicapa (MLP). Adecuados para tareas de clasificación o regresión sobre vectores de características, datasets tabulares, etc.

### **Recurrentes/**
Contiene redes recurrentes, pensadas para datos secuenciales o series de tiempo.  
- `InflacionMX_RNN.ipynb`: Un ejemplo concreto en que se usa una RNN, posiblemente para predecir inflación en México o un indicador económico — ideal para ilustrar aplicación de RNN en serie de tiempo.

### **Material de Exposición / Ejemplos (Expo)**
- `Multicapa_Expo.ipynb` y la carpeta `Expo_Multicapa/`: Notebooks diseñados para explicar paso a paso el funcionamiento de redes multicapa: arquitectura, forward, backprop, entrenamiento, etc. Perfectos como material didáctico o para demostraciones en clase.

## Propósito y Uso

Este repositorio sirve como:

- Material de **aprendizaje** y práctica de arquitecturas de redes neuronales (MLP, CNN, RNN).  
- **Ejemplos prácticos** para experimentar con distintos tipos de redes y tareas (clasificación, predicción, series temporales, etc.).  
- Base para **proyectos o tareas extendidas**: puedes partir de los notebooks existentes para crear nuevos modelos, modificar arquitecturas o aplicar a nuevos datos.  
- Fuente de **documentación y referencia** sobre conceptos de redes neuronales vistos en clase.
